# CHANGELOG

## v1.1.0

- Added environment-driven option overrides through `kc_ngram_options_load_env()`.
- Added `kc_ngram_options_free()` to the public API.
- Added signal listener lifecycle: `kc_ngram_on_signal()`, `kc_ngram_raise_signal()`, `kc_ngram_listen_signals()`, `kc_ngram_listen_signal()`, and `kc_ngram_signal_listener()`.

## v1.0.0

- Published the stable baseline release.
- Provided descending sliding-window n-gram traversal with span closing.
- Supported configurable tokenization and per-chunk command execution.
